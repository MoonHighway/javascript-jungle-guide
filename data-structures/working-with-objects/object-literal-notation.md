# Objects and Literal Notation

(1) Object Literal Notation

```javascript
const agenda = {
  name: "JavaScript Jungle",
  maxParticipants: 20
};

console.log(agenda);
```

(2) Organizing Data with Arrays and Object Hierarchy

```javascript
const agenda = {
  name: "JavaScript Jungle",
  maxParticipants: 20,
  topics: ["browser", "node", "http"],
  instructor: {
    name: "Alex Banks",
    hometown: "Tahoe City"
  }
};

console.log(agenda);
```
